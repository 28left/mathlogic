Recursive and Partially Recursive Functions

An example of a computable, but not p.r. function is the

The Ackermann Function

The Ackermann function is a 2-ary function that is recursively defined by the following equations:

\[ \begin{aligned} A(0,y) &= y+1\\ A(x+1,0)&= A(x,1)\\ A(x+1,y+1)&= A(x,A(x+1,y)) \end{aligned} \]

This is a double recursion, yet one can easily verify that each value \(A(x,y)\) is determined by finitely many “earlier” values \(A(u,v)\) with \(u<x\) or \(u=x \wedge v<y\). Writing \(F_n(m)\) for \(A(n,m)\), we obtain:

\[ \begin{aligned} F_0(m) &= m+1\\ F_1(m) &= m+2\\ F_2(m) & \approx 2m\\ F_3(m) & \approx 2^m\\ F_4(m) & \approx 2^{2^{.^{.^{.^{2}}}}} \text{ ($m$ times)} \end{aligned} \]

This means that the Ackermann function iterates the basic arithmetic operations and grows very rapidly, particularly in the first argument (which indicates the number of iterations). In fact, it majorizes every p.r. function: If \(f\) is a k-ary p.r. function, then there exists a number \(n\) with

\[ f(x_1,\ldots,x_k) \le F_n(\max(x_1,\ldots,x_k)) \]

for all \(x_1,\ldots,x_k\). Thus the 2-ary function \(A\) cannot be p.r. (although on the other hand all \(F_n\) are p.r.!). That the Ackermann function is nevertheless computable can be seen from the fact that it can be obtained from a p.r. function by means of

Minimalization

R6: \(f(\vec{x}) \simeq \mu y \; (g(\vec{x},y)\simeq 0)\),

that is,

\[f(\vec{x}) = \begin{cases} \text{the smallest } y \text{ such that} & \\ \qquad (i) \; g(\vec{x},z)\; \text{is defined for all} \; z\le y \; \text{and} \\ \qquad (ii) \; g(\vec{x},y)=0, \qquad \qquad \quad \text{if such a } y \text{ exists}, \\ \text{undefined} \qquad \qquad \qquad \qquad \; \text{otherwise}. \end{cases}\]

This principle can lead from total to partial functions (e.g., for \(g(x,y)=|x-y^2|\)).

The class R (or \(\mathcal{C}\)) of all recursive functions is the smallest class of functions that contains the initial functions and is closed under substitution, primitive recursion and minimalization. One sometimes also speaks of general recursive, \(\mu\)-recursive functions or (based on Church’s thesis) of computable functions (\(\mathcal{C}\) = computable functions). We will speak of recursive functions primarily in the case of total computable functions.

Thus every p.r. function is also a recursive function (while the Ackermann function is an example of a recursive, but not p.r. function), and the closure properties from previous sections (suitably modified for partial functions) also hold for recursive functions.

Definition: Primitive Recursive and Recursive Relations

Definition 1
A number-theoretic relation \(R \subseteq \mathbb{N}^k\) is called primitive recursive or recursive (or also: decidable) if and only if its characteristic function \(c_R\) is primitive recursive or recursive, respectively, where

\[c_R(\vec{x}) = \begin{cases} 1 & \text{if } \; R(\vec{x}), \\ 0 & \text{otherwise}. \end{cases}\]

Thus, for example, the predicates $; x=y,, x y, , x y, , x<y ; $ are primitive recursive. (Further examples are contained in later chapters.)

Definition 2
A relation \(R \subseteq \mathbb{N}^k\) is called recursively enumerable (r.e. = recursively enumerable, in more recent literature also c.e. = computably enumerable) if and only if it is the domain of a computable function:

\[R \; \text{r.e.} :\iff R = \{\vec{x}\mid f(\vec{x})\downarrow \} \; \text{ for a computable function } \; f.\]

Remarks

  1. A non-empty subset \(A \subseteq \mathbb{N}\) is r.e. if and only if it is the range of a recursive function:

    \(A \, \text{r.e.} \iff A = \emptyset \vee A = \{f(x)\mid x \in \mathbb{N} \} \, \text{ for a recursive function } \; f: \mathbb{N} \to \mathbb{N}\)

  2. In the case of a decidable relation \(R\), there exists an effective procedure that answers the question of whether \(R\) holds or not. A recursively enumerable set \(A\) is also called positively decidable, since there is at least a procedure that can output those numbers that are in \(A\) as elements in a list.   3. It holds that:

    \(A \subseteq \mathbb{N}^k\) is decidable if and only if \(A\) and \(\mathbb{N}^k - A\) are r.e.

  4. There exist r.e. sets that are not recursive; we will encounter examples of these later.
